# AstroKampüs Web Projesi 

Bu proje, Astronomi meraklıları için hazırlanmış modern ve responsive bir web arayüzü tasarımıdır. 

## 🌐 Canlı Demo
Projenin canlı halini buradan inceleyebilirsiniz:
 https://r-a743.github.io/astro-kampus-projesi/

##  Kullanılan Teknolojiler
- **HTML5:** Yapısal içerik düzeni.
- **CSS3:** Modern ve özgün tasarım, gece mavisi uzay teması.
- **Bootstrap 5:** Duyarlı (Responsive) tasarım altyapısı.
- **FontAwesome:** Görsel zenginlik için ikon kütüphanesi.

##  Proje Hakkında
* **Tasarım Dili:** Koyu uzay mavisi arka plan (`#0b0f19`) ile turuncu vurgu renklerinin (`#ff6b35`) uyumu üzerine kurulmuştur.
* **Özellikler:** - Tam uyumlu (Responsive) navigasyon menüsü.
    - Etkileşimli kart tasarımları.
    - Özel turuncu/mavi buton kombinasyonları.
